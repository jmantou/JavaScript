Level II - Ex 31 Variable Scope

In general when the scope of the variable in JavaScript refers to the portions of the program where it can be directly accessed. Before ES^ (2015) edition variables intriduced with the var keyword had only Global Scope and Function Scope.

As we will see in the next exercise JavaScript ES6 introduced Block Scope.Variables declared inside a { } block cannot be aqccessed from outside the block. Variables declared with the var keyword cannot have block scope.
