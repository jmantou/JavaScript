A Regular Expression is a powerful mini language for searching, matching and replacing text patterns. It migrated for the UNIX operating system to many programming languages. Essentially what it does is alloe the programmer to search for complex patterns and expressions within a string.

We usually use Regular Expressions with string match(), replace() methods.
