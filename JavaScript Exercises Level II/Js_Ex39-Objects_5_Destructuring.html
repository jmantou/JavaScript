Level II - Ex 39 Objects 5 Destructuring

Object Destructuring

Destructuring assignment allows you to locally scope fields of an object and declare which values will be used. We can scope some of the keys of an object locally and use them as variables.

Array Destructuring

We can also destructure values from arrays (they are objects after all). We can also pass over unnecessary array values with 'list matching' using commas. That means that commas will take the place of elements that should be skipped.
