Level II - Ex 37 Objects 3 Comparing and Cloning Objects

Comparing Objects

Since equality operators == or === work solely on primitive values, you cannot use them to compare objects because they do nothing more than compare the reference contained to those objects. If you want to examine if two objects contain the same members, you have to do a careful inspection of each and look at each member individually with a loop.

Cloning Objects

Simply assigning objects to another variable does not make a complete duplicate of that object. If you want to clone it (that is make a disconnected copy or deep copy) you have to go to considerable effort to make sure that every member is individually copied from one object to another.
