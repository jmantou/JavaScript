Level II - Ex 36 Objects 2

Objects are passed around by reference and they are never copied. When we make copies of them, we really just copying the references to that object. In Javascript objects can be handled almost interchangeably like arrays. You can access the mebers of an object (methods or properties) as if they were key/value pairs ina a Java Hash Table, Python Dictionary or PHP Associative array.

Prototyping

Every object is linked to a prototype object from which it can inherit properties. All objects created from object literals are linked to Object.prototype that comes as the archetypal Javascript object.

Iterating

You can iterate over the properties of a JavaScript object with the for ... in loop. This is even true for the global browser object or any DOM nodes as well. We can use thios technique to iterate over objects for the purpose of comparing them, displaying them, sorting them or copying them (deep copies).
