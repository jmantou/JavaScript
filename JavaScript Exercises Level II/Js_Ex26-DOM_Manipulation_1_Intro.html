Level II - Ex 26 - Document Object Model (DOM) 1

The DOM is your gateway to the contents of a web page and the key to making your pages come alive with JavaScript. The DOM is also an extremely detailed resource from which you can determine just about anything about the content and layout of a web page.

The easiest way to find an HTML element in the DOM, is by using the id element. If the element is found, the method returns the element as an object you can assign to a variable, otherwise element is null.

If we want to find all elements with the same tag we can use getElementsByTagName() method.

If we want to find all HTML elements with the same class name we can use getElementsByClassName() method.

If we want to find all HTML elements that match a specified CSS selector (id, class, types) we can use the querySelectorAll() method.
