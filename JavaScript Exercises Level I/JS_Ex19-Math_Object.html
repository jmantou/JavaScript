JavaScript Exercise 19 - The Math Object

The Math object is a powerful static object (no constructor) that supplies us with methods for performing various mathematical tasks. The object is static which means that we can use it's methods and properties without creating a Math object first.

You can call a method of an object by writing it's name followed by a dot(.) and the method name. In parentheses following the method name you must include the method argument.
