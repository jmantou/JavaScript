JavaScript Exercise 21 - The Date Object

The Date object is a powerful object we can use to store precise time values down to the millisecond and translate between dates in different locales. We can also find the current date and time, store our own dates and times and do calculations with these dates down to the millisecond.

To create a Date object we have to use the new operator. There is no literal date format so we have to use the constructor (new keyword). Whenever we create an empty instance, it reflects the exact moment at which it was created. You can use this technique to display current date and time on your web page.
