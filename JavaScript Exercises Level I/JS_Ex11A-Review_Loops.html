JavaScript Exercise 11A - Loops v2

Whenever we want a specific piece of code to run many times to run many times, or want to work sequentially on each element of an array or object we use loops. Loops most of the time run until a condition is met.

There are several loop types available depending on how and when we want the condition to be checked. Generally it's better to use the loop that apllies best to the programming solution we try to achieve.
