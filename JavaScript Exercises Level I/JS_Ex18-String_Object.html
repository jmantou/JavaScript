JavaScript Exercise 18 - String Object & Methods v1

JavaScript strings are one of the most important components of the language, probably used more than any other data type. Though we may get numeric values from forms, the values are retrieved as strings, which then are converted to numeric values.

String Literals & String Objects

A string in JavaScript can be both a primitive data type or an instance of the String object. Most of the time these two formats are interchangeable although there are small differences depending on the use.

Strings are immutable. All string methods return a new value. They do not change the original variable.
