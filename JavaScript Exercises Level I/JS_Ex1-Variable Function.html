Level I - Ex 1 - Declaring and Using Variables

Variables is a core feature of every language. They are containers for values that occupy different memory locations. In Javascript we declare variables with the var keyword.

It is best to give meaningful names to our variables, for better code understanding. Console.log() logs in the browser console the value of the variables.

var a = 12; // var number is better
var b = "John"; // var name is better
var c = true; // vas hasValue maybe better
console.log(a);
console.log(b);
console.log(c);

console.log("Type of " + a + " " + typeof a);
console.log("Type of " + b + " " + typeof b);
console.log("Type of " + c + " " + typeof c);
