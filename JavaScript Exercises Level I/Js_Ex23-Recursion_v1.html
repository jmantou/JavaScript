Recursion is a basic programming technique in which a function calls itself repeatedly to solve a problem. Many programming problems and algorithms can be solved this way and some problems that can be solved by other techniques are better solved by recursion.

The concept of recursion can be difficult at times and many programmers try to avoid it if they can. But some programming problems can be solved very efficiently with recursion and with more cost-effective algorithms.

Calculating the Factorial of a Number

The calculation of a number factorial is a classic example for introducing recursion. The Factorial of any given integer n is the product of all the integers from 1 to n. Thus, the factorial of 5 is 120.

5! = 1 * 2 * 3 * 4* 5 = 120
