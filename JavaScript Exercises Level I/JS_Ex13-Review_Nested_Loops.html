JavaScript Exercise 13 - Nested Loops

All the loops we learned so far can be nested. For every run of the outer loop the nested loop repeats itself the necessary times. It is best for the programmer to avoid nesting more than three levels because it makes the code exceedingly difficult and error prone.

The Multiplication Table

In this exercise we use two nested for loops to produce The Multiplication Table. We also use some HTML inside JavaScript to format the result as a table.
