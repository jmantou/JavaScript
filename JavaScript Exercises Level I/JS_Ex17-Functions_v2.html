JavaScript Exercise 17 - Functions Part II

Very often we can have our function return a value and assign that value to an identifier for further use.

Returning Values From Functions

All functions return a value. We can make the function return the value we want using the return statement. If no return statement is specified, an automatic value of undefined is turned instead. Whenever function code reaches return, function execution stops even if there is more code to be executed in the function.

Variable Scope in Functions

The scope of a variable refers to the portions of the program where it can be directly accessed. In JavaScript variables can have global, block or function scope depending on the way we declare them. We will examine further the concept of scope in later exercises.
