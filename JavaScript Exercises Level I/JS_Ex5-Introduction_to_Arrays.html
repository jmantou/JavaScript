Level I - Ex 5 - Introduction to Arrays

In most mainstream languages an array is a linear allocation of memory in which elements are accessed by integers as indexes. Arrays can be very fast and efficient data structures.

Unfortunately this is not the way arrays work in JavaScript. Here arrays are like objects with integer keys and they can have values of any kind. They are slower than the classic arrays but can be used with more convinience in some situations.

JavaScript allows an array to contain any mixture of values. Arrays inherit useful methods from array object that can be used upon them. In JavaScript arrays indexes always start from '0'.
