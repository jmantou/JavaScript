JavaScript Exercise 10 - Do .. . While Loops

Whenever we want a specific piece of code to run many times, or want to work sequentially on each element of an array or object we use loops. Loops most of the time run until a condition is met. This is especially the case with do ...while loops that has some differences with simple while loops.

The imporatant difference between while and do ...while loops involves how the Boolean expression that controls the loop is checked. With a do ...while statement, the body of the loop is executed first and the Boolean expression is checked after the loop body is executed.

Thus, the do ... while statement always executes at least once. After this these two types of loops behave the same. After each iteration of the body of the loop, the expression is checked again, and if is true, the loop continues. If it has changed from true to false the loop statement ends.

We also introduce in this loop the break and continue statements that can be use with all kinds of loops and block statements.
